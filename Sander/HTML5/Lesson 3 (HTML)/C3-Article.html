The New Elements

New HTML5 tags make Web page and application development easier than ever. One of the very handy new features of HTML5 is the use of sementatic markup.

Semantic markup

Gives better meaning, or definition, to tags so they make more sense to humans, programs, and Web browsers.

Not all HTML tags have been replaced or updated for HTML5, but some new tags introduced in HTML5 make the work of creating Web pages a lot easier.
